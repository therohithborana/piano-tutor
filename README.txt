# Musical-Fingers-Project

INSTRUCTIONS TO RUN MUSICAL-FINGERS-PROJECT:

1. Install Python Libraries PyGame, MediaPipe, and cv2.

2. Copy and Paste Audio Files (C.wav D.wav etc...) in your working directory.

3. Copy Musical_Fingers.py file.

4. Run the Code.

5. Two Modes Available:
   - Free Play: Contract fingers to play notes
   - Tutorial Mode: Follow the visual markers to play songs
     * Currently supports "Happy Birthday"
     * Green circle indicates the next note to play
     * When you play the correct note, it automatically moves to the next one

6. Play a Song contracting respective fingers.
